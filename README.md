## Getting started 

Prerequisites

- Git
- If doing work for Android, see [Android development setup](docs/android-development-setup.md)

```pwsh
git clone https://github.com/wwu-cs450/Habit-Task-Tracker.git
cd Habit-Task-Tracker
```

To check if notifications are working, run with `flutter run --dart-define=NOTIF_TEST=true`. You should see notifications on app load, 10s after load, and 20s after load.
